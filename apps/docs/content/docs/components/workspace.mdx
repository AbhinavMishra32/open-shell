## Slot and bottom panels

`SlotPanel` is the generic tab system. `BottomPanel` adds resizing and the expected lower-drawer geometry.
Inactive content can remain mounted so terminals, inspectors, and long-running tools keep their state.

<ComponentPreview name="slot-panel" />

## Thread and file surfaces

`ThreadSurface` renders conversation and agent activity rows. `FileBrowserPanel` composes file tabs,
breadcrumbs, a caller-provided editor, and a side panel. `TerminalSurface` provides a styled host for output;
applications can mount xterm or another terminal engine inside the panel system.
