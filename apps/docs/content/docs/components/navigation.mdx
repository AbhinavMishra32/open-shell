## Sidebar

`Sidebar` composes projects, threads, app navigation, custom sections, scrolling content, and footer actions.
Use `SidebarPrimary`, `SidebarScroll`, and `SidebarFooter` when you need lower-level composition.

## File tree

`FileTree` accepts a nested caller-owned model, searchable presentation, selection, Git status, and optional
action lanes. The sidebar variant removes unnecessary container treatment.
