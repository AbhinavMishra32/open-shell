title: Settings
description: A complete settings navigation and content grammar for desktop and web applications.
---

The settings system includes `SettingsSidebar`, `SettingsPanel`, `SettingsSection`, `SettingsCard`,
`SettingsRow`, `SettingsToggle`, `SettingsSelect`, and `SettingsOptionCard`.

<ComponentPreview name="settings" />

Use rows for compact binary and option controls. Use option cards when a decision needs explanatory copy or
visual comparison. Keep inaccessible sections out of the navigation model rather than presenting disabled
destinations.
