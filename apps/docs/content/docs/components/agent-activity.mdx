---
title: Agent activity
description: Compact, expandable activity and suggestion primitives for agentic product flows.
---

`AgentActivity` keeps active work to one calm line, then reveals individual operations when the user asks
for detail. It is designed for compiler passes, research, tool calls, validation, and other multi-step agent
work without turning the surrounding product into a transcript.

<ComponentPreview name="agent-activity" />

Use `AgentThinking` for a single expandable thought or reasoning summary. Use `AgentSuggestion` for a
focused, caller-owned action after the activity completes. Labels, details, icons, actions, and status mapping
remain application-owned so the system works outside chat products.
