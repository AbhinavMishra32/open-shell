## Button

`Button` supports primary, secondary, soft, ghost, and danger roles. `IconButton` uses the same interaction
contract with a stable square footprint. `Pill` and `StatusDot` cover compact metadata.

## HoverPreview

`HoverPreview` portals contextual content beside its trigger, clamps to the viewport, repositions on scroll,
and supports keyboard focus. Use it for project summaries, symbol context, knowledge snippets, and metadata.
