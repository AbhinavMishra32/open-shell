`AppShell` coordinates chrome, sidebar, header tabs, main content, composer, right panel, and bottom panel.
Each region is a slot. Opaline owns geometry and interaction affordances; your app owns content and state.

<ComponentPreview name="app-shell" />

## Composable anatomy

- `AppShellChromeControls` and `AppShellSidebarChrome` compose window controls.
- `AppShellHeader`, `AppShellTabStrip`, and `AppShellHeaderActions` compose top navigation.
- `AppShellContent`, `AppShellComposer`, `AppShellRightPanel`, and `AppShellBottomPanel` expose layout regions.
- The high-level `AppShell` prop API remains useful when the standard arrangement is enough.
