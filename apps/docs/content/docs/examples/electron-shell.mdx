---
title: Electron shell
description: Run the desktop example that consumes @opaline/ui as a workspace package.
---

From the repository root:

```bash
npm install
npm run start:example
```

The example demonstrates hidden inset titlebar behavior, sidebar chrome, shared history controls, app shell
regions, file navigation, composer controls, and bottom-panel tools.

Its package dependency resolves to the local `@opaline/ui` workspace during development and to the same npm
package boundary used by external applications.
