---
title: Local development
description: Run the documentation against live Opaline component source.
---

From the repository root:

```bash
npm install
npm run docs:dev
```

Open [http://127.0.0.1:3011](http://127.0.0.1:3011).

The docs TypeScript path maps `@opaline/ui` to `packages/ui/src/index.ts`, and the root layout imports the
source stylesheet. Editing a component or token therefore updates its docs preview through Next.js HMR.
No package publication or `@opaline/ui` build is required during documentation work.

Run focused validation without a production build:

```bash
npm run typecheck -w @opaline/ui
npm run typecheck -w @opaline/docs
```
