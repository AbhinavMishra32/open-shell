# Opaline Electron Example

Runnable Electron app that consumes `@opaline/ui` from the workspace.

## Run

```sh
npm install
npm run start:example
```

## What It Demonstrates

- Transparent Electron window surface.
- Hidden inset macOS titlebar.
- Translucent sidebar material.
- Chat/thread canvas.
- Composer dock.
- Radix-backed dialog and dropdown primitives.
- Bottom terminal panel.
- File tree side content.

This example intentionally uses the readable library package instead of bundled research internals.
